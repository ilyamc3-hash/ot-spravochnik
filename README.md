# ОТ Навигатор

Личный справочник по охране труда: газоопасные и огневые работы, обучение и
инструктажи, РПО (общие требования), АПК. Структурированные пункты СТО с
поиском, избранным и чеклистом видеофиксации РПО.

Собран на React + Vite. Данные и вся логика — в `src/App.jsx`.

## Разработка

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`.

## Деплой на GitHub Pages

Проект уже настроен на автодеплой через GitHub Actions
(`.github/workflows/deploy.yml`): каждый `git push` в ветку `main` собирает
проект и публикует его на GitHub Pages.

### Первоначальная настройка (один раз)

1. Создай пустой репозиторий на GitHub с именем **`ot-spravochnik`**
   (если назовёшь иначе — поправь `base` в `vite.config.js` и `start_url`/`scope`
   в `public/manifest.json` на `/твоё-имя-репозитория/`).

2. Запушь код:
   ```bash
   git remote add origin https://github.com/ilyamc3-hash/ot-spravochnik.git
   git push -u origin main
   ```

3. В настройках репозитория на GitHub: **Settings → Pages → Source** — выбери
   **GitHub Actions** (не "Deploy from a branch").

4. После первого пуша зайди во вкладку **Actions** — там будет виден прогресс
   сборки и деплоя. Через минуту-две сайт будет доступен по адресу:
   ```
   https://ilyamc3-hash.github.io/ot-spravochnik/
   ```

### Дальнейшие обновления

Просто `git push` — деплой запускается автоматически.

## Установка как приложение (PWA)

На телефоне: открой сайт в браузере → меню → «Добавить на главный экран».
Иконка появится как у обычного приложения, без адресной строки браузера.

## Структура данных

Все требования, этапы и разделы (ГОР, ОР, Обучение, РПО, АПК) заданы как
JS-объекты в начале `src/App.jsx`. Чтобы добавить новый пункт — найди нужный
массив `*Detail` и добавь объект `{ label: "...", ref: "..." }` в список
`items` нужной группы.
